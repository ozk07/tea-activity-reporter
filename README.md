# tea-activity-reporter
A FastAPI backend application for fetching cryptocurrency data from multiple exchanges and providing comprehensive analysis.

## Features
- 🔗 Multi-exchange support (Binance, Bybit, OKX, Coinbase)
- 📊 Real-time price data fetching  
- 📈 Advanced klines analysis and statistics
- 🚀 Fast and scalable FastAPI backend
- 🎯 Standardized data format across exchanges

## Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Run the API server
cd scripts
python app.py
```

Visit http://localhost:8000/docs for interactive API documentation.

For detailed API usage, see [API_README.md](API_README.md).
